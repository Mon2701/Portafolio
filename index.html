GAIA

Este proyecto desarrolló una página web interactiva que opera como una aplicación cliente-servidor. Se empleó PHP como lenguaje de servidor, y HTML, CSS y Bootstrap para la interfaz de usuario y la presentación del contenido. La página web se conecta directamente a una base de datos, posibilitando que los usuarios interactúen con la información almacenada.

  • Creada - 22 Septiembre 2022
  • Tecnologias usadas - html, CSS, Boostrap y PHP
  • Roles - Frontend

Mini barco

Junto con compañeros de la universidad construimos una pequeña embarcación controlada por un microcontrolador Arduino. Utilizando motores, sensores y una placa Arduino programada para controlar la dirección, la velocidad y otras funciones, en donde aprendimos un poco de electrónica y programación.

  • Creada - 10 Marzo 2022
  • Tecnologias usadas - Arduino
  • Roles - Programación y armar circuito

Pollito Pio

Junto con compañeros de la univerdad desarrollamos una página web que cuenta con un sistema de logs para permitir el monitoreo y la auditoría de las actividades en un sistema o aplicación y la verificación de campos calidad de los datos y con implemmentación de SQL injection para reducir el riesgo de errores y asegurar la confiabilidad.
